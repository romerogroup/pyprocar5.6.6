How to cite
============

If you have used PyProcar in your work, please cite:

`Uthpala Herath, Pedram Tavadze, Xu He, Eric Bousquet, Sobhit Singh, Francisco Muñoz, and Aldo H. Romero. "PyProcar: A Python library for electronic structure pre/post-processing". Computer Physics Communications 251 (2020): 107080. <https://www.sciencedirect.com/science/article/pii/S0010465519303935>`_

Thank you.
